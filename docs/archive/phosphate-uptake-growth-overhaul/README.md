# Phosphate uptake and growth overhaul

These files preserve the planning, test-driven implementation, and verification
history of the phosphate uptake and growth overhaul. Current scientific
behaviour is described in [`../../phosphate-model.md`](../../phosphate-model.md),
current code ownership in [`../../module-map.md`](../../module-map.md), and
unimplemented decisions in [`../../open-questions.md`](../../open-questions.md).

The original execution used short identifiers such as `P0`–`P7`. Descriptive
filenames and headings now identify each subject; legacy task IDs remain inside
the historical records so old test evidence and diary entries remain
interpretable.
