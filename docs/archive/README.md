# Archived implementation records

These files preserve the planning, test-driven implementation, and verification
history of the phosphate model. They are non-authoritative: current scientific
behaviour is described in [`../phosphate-model.md`](../phosphate-model.md),
current code ownership in [`../module-map.md`](../module-map.md), and unresolved
decisions in [`../open-questions.md`](../open-questions.md).

The original execution used short identifiers such as `P0`–`P7`. Descriptive
filenames and headings now identify the subject of each record; legacy task IDs
remain inside the historical text so old test evidence and diary entries can
still be interpreted.
